## Projetos em Destaque

### API de Monitoramento de Serviços  
GitHub: https://github.com/Linzackk/monitoramentoServicos  
Demo: https://monitoramentoservicos.onrender.com/docs/  

API desenvolvida em Node.js e Express que permite cadastrar serviços para análise automática a cada 15 minutos.  
O sistema verifica o status das aplicações e registra informações sobre falhas, instabilidades e possíveis incidentes com base nas respostas dos serviços monitorados.

---

### Minha Biblioteca  
GitHub: https://github.com/Linzackk/MinhaBiblioteca  
Demo: https://minha-biblioteca-wine.vercel.app  

Aplicação web em React que permite aos usuários pesquisar, salvar e acompanhar o status de livros que estão lendo.  
O projeto foca na organização pessoal de leitura e consumo de APIs externas.

---

### Pokedex  
GitHub: https://github.com/Linzackk/pokedexReact  

Aplicação em React que consome a PokeAPI para permitir a busca e visualização de informações sobre pokémons.  
O projeto trabalha integração com API, componentes e organização de interface.

---

### Sistema Acadêmico  
GitHub: https://github.com/Linzackk/PIM-2oSemestre-Unip  

Sistema em console desenvolvido em Python que simula um ambiente acadêmico.  
Permite o cadastro de alunos, professores e coordenadores, além do acompanhamento de notas, faltas e dados pessoais utilizando conceitos de lógica e POO.
