## Projetos em Destaque

### Sistema de Treinos de Academia
GitHub: https://github.com/Linzackk/TreinosAcademia

API REST desenvolvida em **C# com .NET**, utilizando **Entity Framework Core** e **SQL Server**.  
Gerencia usuários, treinos e exercícios com relacionamentos N:N, aplicando DTOs, AutoMapper, arquitetura em camadas e consultas otimizadas com `AsNoTracking`, `Include` e `ThenInclude`.

---

### API de Monitoramento de Serviços
GitHub: https://github.com/Linzackk/monitoramentoServicos  
Demo: https://monitoramentoservicos.onrender.com/docs/

API desenvolvida em **Node.js e Express** que monitora serviços automaticamente a cada 15 minutos.  
Verifica disponibilidade, registra falhas e instabilidades com base nas respostas dos serviços monitorados.

---

### Minha Biblioteca
GitHub: https://github.com/Linzackk/MinhaBiblioteca  
Demo: https://minha-biblioteca-wine.vercel.app

Aplicação web em **React** para pesquisar, salvar e acompanhar o status de leitura de livros.  
Foca em consumo de API externa e organização de dados do usuário.
